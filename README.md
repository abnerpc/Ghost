# Ghost [![Build Status](https://travis-ci.org/vsouza/Ghost.svg?branch=master)](https://travis-ci.org/vsouza/Ghost)

Manipulate your hosts in the simplest way.

## Usage: 

**Add host**

``` ghost add {{NAME}} {{ip}} ```

**Remove host**

``` ghost rm {{NAME}} {{ip}} ```

**Show my hosts**

``` ghost show ```

**Show my active hosts**

``` ghost show active ```

**Show my inactive (commented) hosts**

``` ghost show inactive ```
